getPercentDistanceTraveled

open fun getPercentDistanceTraveled(): Float

Returns the percentage of the distance traveled that was last calculated. This is only calculated if the vanishing route line feature is enabled.

Return
Float: the value calculated during the last progress update event or 0 if not enabled.
